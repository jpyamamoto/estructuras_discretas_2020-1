En esta práctica hicimos uso de recursión para ejecutar ciertas acciones.
Pudimos ver que la recursión se usa principalmente en listas, para realizar algo varias veces, hasta que encontremos un caso base, con lo cual termina la ejecución de la función.
Para la función isort, también hicimos uso de la función delete en Data.List. Esta función sirve para eliminar la primer ocurrencia de un cierto elemento dentro de una lista.
