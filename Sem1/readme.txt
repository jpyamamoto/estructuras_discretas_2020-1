Semanal 1
Desarrollamos esta práctica haciendo uso de los operadores básicos de Haskell que vimos durante la clase para implementar en código ciertas fórmulas matemáticas.
También hicimos uso de if..then..else para la toma de ciertas decisiones.
Además, utilizamos patrones para realizar distintas acciones dependiendo de los parámetros que recibe la función.
